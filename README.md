# 随机选择.php

![php](https://img.shields.io/badge/php->=5-blue.svg)![build](https://img.shields.io/badge/build-passing-4BC51D.svg?style=flat)

一个用 PHP 写的随机选择页面。

### 1. 摄像头功能

除了是用预设静态资源外，你还可以使用 `/camera` 下的组件，调用设备的摄像头，并进行实时录入照片，进行随机选择，适用于抽奖活动等。
